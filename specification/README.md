SMART on FHIR Protocol
---------------------------------------------------

This draft document is an attempt to extrapolate/define a formal protocol between SMART applications and electronic health records from the 
[argonaut-dev branch](#https://github.com/smart-on-fhir/smart-on-fhir.github.io/tree/argonaut-dev) of the SMART on FHIR technical documentation.
It is currently included for collaborative purposes, and should not be considered authoritative.
